#### Why Astro?
Astro was the first framework to support the [View Transitions API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API), the ability to animate elements across a Multi-Page Application. I still find it incredibly impressive every time I come back to the site. 
<br>
<br>
Astro is primarily a Static Site Generator, and ships only HTML and CSS by default. This makes for an incredibly snappy browsing experience on simple websites. In a world of sometimes barely functioning SPA's it was a specific goal of mine to make something quick. 
<br>
It also has first party support for handling and styling blog content, like this Markdown file, which makes it an appropriate choice for a portfolio.
