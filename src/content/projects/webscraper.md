#### Web scraper
I wanted to write a webscraper due to the utility of having a personalised one ready for future projects. 

In the end it I ended up writing more html parsing initially, and parsing html directly from the WikiMedia API.
I also ended up writing a second version of this repo to automate getting and swapping Advent of Code input, which proved useful as the week after was the start of December.


#### Learnings 
This project taught me a lot about web security, allowing me to be more confident about concepts like the robots.txt file, authenticating using tokens, and navigating external Rust dependencies.
